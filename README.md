# Money Monster
Solidity Smart Contract & dApp made for Money Monster

## Solidity Smart Contract
A BUSD staking protocol with features such as:
- Progressively increasing APR
- Fees
- 100% decentralized (no owner functionality)
- Referral system

**Audited by The Stamp:** [Audit Link](https://web.archive.org/web/20230226061747/https://the-stamp.com/2023/01/money-monster/)

## The dApp
Built with the old but golden trio: HTML, CSS, JS (web3js, jQuery).

**Live Link:** [moneymonster.io](https://web.archive.org/web/20230202163644/https://moneymonster.io/)
